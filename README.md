# pomodoro-dashboard-app🍅

A clean and aesthetic Pomodoro Timer web app designed to improve focus and productivity.

## 🚀 Features

- ⏱️ Pomodoro timer (Focus, Short Break, Long Break)
- ▶️ Play / Pause / Reset controls
- ⚙️ Custom timer settings (user-defined durations)
- 🔊 Sound toggle (on/off)
- 🎨 Modern UI with glassmorphism design
- 📱 Responsive layout (in progress)

## 🧠 Upcoming Features

- 📅 Calendar integration
- ✅ To-Do List management
- 📊 Productivity Dashboard
- 📈 Live analytics (focus sessions, trends)
- 🔔 Notifications & reminders

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Glassmorphism UI)
- JavaScript (Vanilla)
